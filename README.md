# metronome-detector
README.md
Metronome Detector
메트로놈 박자에 맞춘 기타 연주의 리듬 정확도를 분석하는 도구입니다. 실시간 녹음 데이터를 기반으로 정박 지점과의 오차를 계산하고 시각화된 리포트를 제공합니다.

주요 기능
저지연 녹음 및 재생: ASIO 드라이버를 통해 입력 지연을 최소화하며 메트로놈과 동기화된 녹음을 진행합니다.

상태 기반 피크 탐지: 신호의 포착 지점을 정확히 찾기 위해 정적 상태 확인 후 피크를 감지하는 알고리즘을 사용합니다.

리듬 정확도 시각화: 연주된 음이 정박 그리드에서 얼마나 벗어났는지 자동으로 판별하여 표시합니다.

이미지 리포트 생성: 분석 결과 파형과 오차 지점이 기록된 이미지를 자동 저장합니다.

설치 및 요구 사항
Python 3.11 이상

필수 라이브러리: numpy, sounddevice, matplotlib

ASIO를 지원하는 오디오 인터페이스 권장

사용 방법
config.py 파일의 ASIO_DEVICE_ID를 본인의 오디오 장치 번호에 맞게 수정합니다.

분석하고자 하는 METRONOME_BPM과 CHROMATIC_BEATS(분할 박자)를 설정합니다.

main.py를 실행합니다.

안내에 따라 카운트인 이후 연습하려는 크로매틱 패턴을 연주합니다.

녹음이 종료되면 자동으로 분석 결과 그래프가 화면에 출력되고 images 폴더에 파일로 저장됩니다.

분석 결과 해석
Blue Signal: 녹음된 기타 파형입니다.

Red/Pink Line: 메트로놈의 마디 시작과 박자 지점입니다.

Green Dashed Line: 프로그램이 감지한 연주 시작 지점입니다.

Red X: 설정된 TOLERANCE 범위를 벗어난 리듬 오차 지점입니다.
